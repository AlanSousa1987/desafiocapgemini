As soluções foram desenvolvidas na linguagem javascript pois tenho mais afinidade.

Para executar as questões é preciso seguir os seguintes passos:

* Clonar o repositório https://github.com/AlanSousa1987/desafiocapgemini.git
* Instalar o nodejs no endereço:  https://nodejs.org/en/download/
* Executar o arquivo da respectiva questão. Exemplo:

node <Arquivo>.js

É possível também executar o conteúdo de cada arquivo no browser através da ferramenta de desenvolvedores.
